# 🚀 Customer Churn Analysis & Prediction

## 📊 Project Overview

This project focuses on analyzing customer churn behavior and building a machine learning model to predict whether a customer is likely to leave (churn). The goal is to help businesses identify high-risk customers and improve retention strategies.

## 🧰 Tools & Technologies

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

## 🔍 Workflow

1. **Data Cleaning & Preprocessing**

   * Converted `TotalCharges` to numeric
   * Handled missing values using median imputation

2. **Exploratory Data Analysis (EDA)**

   * Analyzed customer behavior patterns
   * Identified trends related to churn

3. **Feature Engineering**

   * Converted categorical variables using one-hot encoding
   * Prepared dataset for machine learning

4. **Model Building**

   * Logistic Regression
   * Random Forest Classifier

5. **Model Evaluation**

   * Achieved ~82% accuracy using Logistic Regression
   * Compared performance with Random Forest


## 📈 Key Results

* Model Accuracy: **~82%**
* Successfully predicted customer churn using supervised learning


## 💡 Key Insights

* Customers with **higher monthly charges** are more likely to churn
* Customers with **shorter tenure** show higher churn probability
* **Month-to-month contracts** have the highest churn risk
* Pricing and contract type significantly impact customer retention
